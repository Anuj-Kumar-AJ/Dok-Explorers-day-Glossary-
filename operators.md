# Operators
Welcome to Operators glossary 

This glossary covers the most important and relevant terms related to operators in k8s, and each term is defined

| Term | Definition |
| --- | --- |
|Custom Resource Definition (CRD)|A Kubernetes object that enables users to define their own API objects and create custom controllers to manage them|
| Kubernetes Operator|A method of packaging, deploying, and managing Kubernetes-native applications as Custom Resource Definitions (CRDs) and associated controllers|
| Operator Framework|A toolkit that provides a set of tools and best practices for building Kubernetes Operators|
| Operator Lifecycle Manager (OLM)|A Kubernetes component that enables users to manage the lifecycle of Operators and their associated CRDs|


We hope this glossary helps you better understand the world of operators in k8s, and enables you to explore these exciting technologies with confidence. If you have any feedback or suggestions for new terms to add to the glossary, please feel free to contribute to the repo
